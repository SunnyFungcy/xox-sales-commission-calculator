# XOX Sales Commission Calculator

DEX 銷售獲益（提成）計算器 — Next.js + TypeScript。支援繁中／英文介面，規則可摺疊參考與編輯（需密碼）。

## 開發

```bash
npm install
npm run dev
```

瀏覽 [http://localhost:3000](http://localhost:3000)。

## 建置

```bash
npm run build
npm start
```

## 技術棧

- Next.js 14 (App Router)
- React 18、TypeScript、Tailwind CSS
